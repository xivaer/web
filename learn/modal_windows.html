In this video i will show you how to create modal windows in Bootstrap. Modal windows are flexible dialog prompts windows with minimum required functionality. Those windows that you see that ask's for surveys, or subscriptions that makes the background little bit black are modal windows.A modal windows consists of a a modal title which uses a modal-header class, than a modal body which uses a modal-body class, than a footer with the button links and whatnot which uses the modal-footer class. We are also going to create little fade in and fade out effect in those created modals, also i will show you how to create small as well as large modals.
